IRN Status Checker

Enter the 64-character Invoice Reference Number to validate its format and get AI-powered compliance guidance.

🔐
What is IRN?
A unique 64-character SHA-256 hash generated by the IRP for each registered e-Invoice.
⏳
IRN Validity
IRN is permanent and cannot be modified. Cancellation only allowed within 24 hours of generation.
🏛️
IRP Portals
6 IRPs authorised: NIC, Clear (formerly ClearTax), IRIS, Cygnet, EY and GSTN.
⚠️ Integration Note: Live IRN lookup requires integration with the NIC IRP API (api.einvoice1.gst.gov.in). This app validates IRN format and provides AI compliance guidance. See the README for full production API integration steps.
🚛

e-Way Bill Status

Enter the 12-digit e-Way Bill number to validate its format and get AI-powered guidance on validity and compliance.

📦
EWB Threshold
Mandatory for goods movement exceeding ₹50,000 in value — both interstate and intrastate.
🗓️
Validity Rules
1 day per 200 km for regular cargo. Minimum 1 day. ODC: 1 day per 20 km.
🔄
Extension Window
EWB can be extended within 8 hours before expiry or 8 hours after expiry by generator or transporter.
⚠️ Integration Note: Live e-Way Bill status requires integration with the NIC EWB API (api.ewaybillgst.gov.in). This app validates format and provides AI guidance. See the README for full production API integration steps.
